# neurons image analysis

[![GitHub Actions CI/CD Status](https://github.com/matthiaskoenig/neurons-image-analysis/workflows/CI-CD/badge.svg)](https://github.com/matthiaskoenig/neurons-image-analysis/actions/workflows/main.yml)

neurons-image-analysis is a collection of python utilities for working with
histology images of neurons with code available from
[https://github.com/matthiaskoenig/neurons-image-analysis](https://github.com/matthiaskoenig/neurons-image-analysis).
